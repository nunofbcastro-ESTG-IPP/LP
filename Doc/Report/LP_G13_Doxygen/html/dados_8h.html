Está função verifica se o utilizador ja tem alguma empresa atribuida e senão pede para a criar, de seguida chama a função "exportarRelatorioFinal"

Parâmetros
empresa- apontador com todos os funcionarios
irsdados- apontador com todos os dados do IRS
nomesEmpresas- apontador com os nomes das empresas(nome das pasta)
taxasContributivas- apontador com as Taxas de Contribuinte
